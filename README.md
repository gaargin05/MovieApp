# MovieApp 

MovieApp is an Android application developed using **Kotlin** and **Jetpack Compose** as part of a Mobile Development assignment.

## Features
- Splash Screen
- Home Screen
- Bottom Navigation:
  - Movies
  - Favourites
  - Watchlist
- Movie list UI
- Movie detail screen UI
- Play button with in-app notification

## Tech Stack
- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Physical Device

## Notes
- Movie data is mocked for demonstration.
- TMDB API integration can be added as a future enhancement.
  
## Assumptions
- The app uses static/dummy data for movies.
- No authentication is implemented as per assignment instructions.
- Internet permission is not required for the current implementation.
